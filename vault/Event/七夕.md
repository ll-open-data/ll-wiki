# 七夕

七夕祭り。春宮ゆくりの呼びかけでいきづらい部のメンバーが短冊に願い事を書いた。佐々木翔音が「仙台の七夕は8月だから！」と指摘。佐々木翔音は七夕に五色そうめんを食べる習慣がある。
