# プログラミングサマーキャンプ

[[麻布麻衣]](attendee) が参加した5日間のプログラミングサマーキャンプ。数字と記号の世界に没頭できることを期待していたが、グループワークやアイスブレーキングタイムなどの交流プログラムが含まれており少し憂鬱だったようす。親への対外的なアピールのために参加した側面もある。

参加中に休憩時間、ふとL高のことやメンバーのことを思い出した。「今までそんなこと思ったことないのに」と感じた初めての経験。
